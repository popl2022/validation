Restoring SC (or not)

  • Simple SC violations
  • With atomic exchange
  • With fences
  • With sfences

First, some “simple” SC violations are selected. By “simple” it is meant that the tests are critical ones, contain no instruction other than memory accesses and have two or three threads. Memory types may vary though.

We then show how to restore SC by using exchange instructions and memory fence mfence instructions, regardless of memory types. We also show that using sfence instructions does not suffice.
